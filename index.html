Questions to ask

  • Is the code open source?
  • Is there a demo?
  • Does the code extend the core or hook into observers?
  • Are the plugin developers responsive?
  • Is the plugin well reviewed?
  • How will the plugin impact the client's workflow?
  • Who developed the plugin?

What Ai looks for

Trusted companies

  • Web store dedicated to their work?
  • Dedicated support system?
  • Responsive to communication?
  • Access developers outside of Magento Connect?
  • Reviews or community interaction?

To plugin, or not to plugin

  • Do not trust Magento Connect, do research
  • Discuss the plugin with your colleagues
  • Ask as many questions as possible
  • Do not modify the plugin code directly
  • Use plugin developers you trust
